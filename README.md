# 赤外線（熱伝導）シミュレーター / IR & Thermal Conduction Simulator

HTML5 Canvas と TypeScript を使用して構築された、簡易的な赤外線照射および2次元熱伝導シミュレーターです。
光源から放射される赤外線（エネルギー）が対象物に吸収され、物質内を熱が伝導していく物理現象をサーモグラフィ風にリアルタイム可視化します。

## 💡 特徴

- **リアルタイム物理演算:** - **逆二乗の法則:** 赤外線の減衰は光源からの距離の逆二乗に比例します。
  - **熱伝導方程式:** 隣接するセル間での熱移動を差分法（2次元格子）で計算。
  - **ニュートンの冷却法則:** 周囲の空気への自然放熱（冷却）をシミュレート。
- **インタラクティブ操作:** マウスカーソルに合わせて赤外線光源がリアルタイムに移動します。
- **直感的な可視化:** 温度変化を青（低温：25℃）〜赤（高温：125℃以上）のカラーマップ（サーモグラフィ風）で表現。

## 🛠️ 開発環境 / 技術スタック

- **言語:** TypeScript
- **描画:** HTML5 Canvas API (外部描画ライブラリなし)
- **ビルドツール:** Vite (Vanilla TS テンプレート)
- **実行環境:** Node.js

🧮 物理パラメータのカスタマイズ
src/main.ts 内の数値を変更することで、シミュレーションの挙動を調整できます。

物質の特性 (Material):
initGrid() 内の iron オブジェクトを変更することで、鉄以外の物質（例：熱が伝わりやすい銅、温まりにくい水など）を再現できます。

TypeScript
const iron: Material = { 
    thermalConductivity: 0.15, // 熱伝導率（高いほど熱が広がりやすい）
    specificHeat: 0.45,        // 比熱容量（低いほどすぐ温まる）
    absorptionRate: 0.85       // 赤外線吸収率（高いほどエネルギーを吸収する）
};
光源の強さ (IRSource):
this.source の intensity（強度）や range（照射範囲）を変更することで、赤外線ランプの出力を調整できます。
